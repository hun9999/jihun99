# jihun-portfolio-site

> AI-assisted portfolio website for designers and creative professionals.  
> Built by a designer — not a developer — using AI tools end-to-end.

---

## Background

As a designer and AI video creator preparing for employment,  
I needed a portfolio site that reflected my actual creative work —  
but traditional dev workflows created a barrier I couldn't cross alone.

This project was built entirely with AI-assisted coding tools  
(Cline + Gemini, Firebase Hosting) with zero prior development experience.  
The result: a fully deployed, interactive portfolio site — built and maintained by a designer.

---

## What This Project Does

A personal portfolio website template optimized for:
- Designers showcasing brand, video, and photography work
- AI-assisted creators who need a deployable site without dev background
- Job seekers in creative fields who want a professional web presence

---

## Key Features

- Interactive Dot Grid canvas effect (custom coordinate-based animation)
- Responsive layout built with Tailwind CSS v4
- Project gallery with category filtering (Brand / Video / Photography)
- Contact page ("HIRE ME") with direct inquiry flow
- Deployed on Firebase Hosting (live URL included)
- AI collaboration credited transparently in footer

---

## Tech Stack

| Tool | Role |
|------|------|
| React + Vite | Frontend framework |
| Tailwind CSS v4 | Styling |
| Firebase Hosting | Deployment |
| Cline + Gemini API | AI-assisted development |

---

## Who This Is For

Designers, photographers, and video creators who want to:
- Own their portfolio site without hiring a developer
- Use AI tools to build and maintain it themselves
- Show employers a live, deployed project — not just a PDF

---

## Impact

- Built and deployed by a non-developer using AI coding tools only
- Demonstrates that creative professionals can ship real web products
- Template is open-sourced so other designers can fork and customize
